# Heritage Vision AI

AI-powered Indian monument recognition system using Transfer Learning with ResNet50.

## Features

- Monument image classification
- ResNet50 transfer learning
- Flask web deployment
- Historical information retrieval
- SQLite monument database
- Confidence score prediction

## Technologies Used

- TensorFlow / Keras
- ResNet50
- Flask
- SQLite
- HTML/CSS

## Monuments Supported

- Taj Mahal
- Charminar
- Gateway of India
- Qutub Minar
- Sun Temple Konark

## Run Project

```bash
pip install -r requirements.txt
python app.py
```
